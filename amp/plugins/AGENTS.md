## Built-in medium mode default rationale

Prefer Amp's built-in `medium` mode for plugin-spawned helper agents and subagents unless there is a specific reason to choose another mode.

Rationale: Amp's July 9, 2026 Dial announcement says `medium` is backed by GPT-5.5 at medium reasoning effort. Chinh prefers this default while `medium` uses OpenAI models because his $100 ChatGPT subscription economics make this materially cheaper in Amp. Reconsider this default if Amp changes `medium` away from OpenAI.
